# Certification Images

Place your certification screenshots here with the following names:

- `ejpt.png` - eJPT (eLearnSecurity Junior Penetration Tester)
- `icca.png` - ICCA (INE Certified Cloud Associate)
- `pmat.png` - Practical Malware Analysis and Triage
- `peh.png` - Practical Ethical Hacking
- `web-security.png` - Web Security certifications
- `rust.png` - Rust 101
- `quantum.png` - Basics of Quantum Information
- `quantum-crypto.png` - Practical Introduction to Quantum-Safe Cryptography

**Recommended image format:** PNG or JPG
**Recommended size:** 800x600 pixels (or similar aspect ratio)

The images will appear as hover previews when you hover over each certification card.
